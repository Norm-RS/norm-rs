# Security Policy

For vulnerability reports, responsible disclosure, compliance-security questions, and commercial licensing requests, contact the NORM maintainers via repository security contact channel.

## Scope

- Cryptographic and audit-chain integrity issues.
- Data leakage or PII exposure paths.
- Regulatory logic bypasses in compliance-critical crates.

## Licensing Note

Regulatory algorithm crates in this workspace use AGPL-3.0-only where declared in crate metadata. Commercial production usage may require separate commercial licensing.
